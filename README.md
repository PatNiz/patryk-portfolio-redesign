# Patryk Niziołek — Portfolio (Redesign)

This is a fully rebuilt, modern, responsive portfolio site generated from the content of your previous page.

## What you need to add (optional)
- `assets/profile.jpg` — your photo (square works best). If missing, the site shows an initials-based placeholder.
- `assets/patrykniziolekCV.pdf` — your CV PDF (the button is already wired).
- Certificate PDFs referenced in `assets/` (optional) — you already had them in your repo, so you can copy them here.

## Local preview
Open `index.html` directly in the browser, or run a tiny static server:

```bash
python -m http.server 8000
```

Then open http://localhost:8000
